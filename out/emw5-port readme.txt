EggmanWay5 for SRB2 v2.2.15 ported by: GLide KS and Ogami_WK

It's recommended that you open the game with the provided batch files to play the mod properly as intended
if you can't run the .bat, open it with notepad and copy the launch parameters and launch SRB2 with those launch parameters.

Tip for Linux users:
- Rename the .bat to .sh
- Edit the .sh files with notepad
- Rename 'srb2win.exe' with SRB2's command line launch 
	Flatpak: 'flatpak run org.srb2.SRB2'
	Compiled: ./path/to/binary_file or just binary_file (Ex: './home/glideks/srb2' | srb2 )
- Give the .sh files execute permissions (through terminal should be 'chmod +x bashfile.sh')
- Run it and have fun!

Read the manual included!

---Recommended Settings---

Render: OpenGL
Models: off
Pallete rendering: off
Shaders: if needed, turn off shaders. some levels has the brightress too low.
Play style: Strafe or Manual
