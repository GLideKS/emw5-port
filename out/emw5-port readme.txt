EggmanWay5 for SRB2 v2.2.15 ported by: GLide KS and Messiah (Ogami_WK)

It's recommended that you open the game with the provided batch files to play the mod properly as intended
if you can't run the .bat, open it with notepad and copy the launch parameters and launch SRB2 with those launch parameters.

Read the manual included!

---Recommended Settings---

Render: OpenGL
Models: off
Pallete rendering: off
Shaders: if needed, turn off shaders. some levels has the brightress too low.
Play style: Strafe or Manual
